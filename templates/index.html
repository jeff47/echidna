Help: Start Page

Use this page to start a new run.

  • Enter the author name (and optional ORCiD).
  • Accepted name formats include: Anthony Fauci, Fauci A, Fauci AS, Fauci, Anthony, and Fauci, A.
  • To look up an ORCiD from the name, click Search ORCiD, review the returned profiles, then click Use ID to auto-fill the ORCiD field (the match list closes after selection).
  • Set year bounds, or check all-years to ignore the year filter.
  • Paste PMIDs or upload a text file with one PMID per line.
  • Choose filtering options (peer-reviewed toggle, expandable default exclusion checklist, and optional extra publication types to ignore).
  • Select Find Author Clusters to continue to author disambiguation.
